## GitHub　Actions
GitHub上にて行われるイベント(Push,Issue,Releaseなど)をトリガーとしてワークフローを起動し様々な動作を自動化することができる機能。  
例えば、一定期間以上放置されているIssueを自動でCloseしたりすることができる。
